# CastCore NDF Baseline

Dieses Paket bindet CastCore dokumentarisch an das Nova Development Framework an.

## Ziel

CastCore erhält eine NDF Project System Baseline mit:

- Project Manifest
- Project Profile
- Capability Matrix
- Compliance Check
- Health Score
- Work Package Queue
- Project Brain
- Risiken
- Lessons Learned
- NDF Workflow
- Quality Gates
- Release Process
- Claude Prompts

## Wichtig

Dieses Paket enthält **keine funktionalen Codeänderungen**.

Es verändert nicht:

- Backend-Code
- Frontend-Code
- Worker-Code
- Dockerfiles
- Docker Compose
- GitHub Actions
- Tests

docs(ndf): add CastCore NDF baseline
