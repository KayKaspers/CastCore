# CastCore – Projektgedächtnis für Claude

## Projektrolle

CastCore ist eine self-hosted Streaming- und Media-Suite.

Ziel ist eine professionelle, Docker-first, anfängerfreundliche Suite für Streamer, Content Creator und Self-Hoster.

## Standard-Workflow

Nova plant.
Claude implementiert.
Cursor prüft.
GitHub ist die zentrale Codequelle.
Nextcloud ist nur Ablage für Prompts, Reviews, Feedbacks, Branding und Exporte.

## Projektstandards

- Docker-first
- Documentation-first
- Security-first
- Deutsch und Englisch
- Anfängerfreundlich
- Modular
- Simple Mode und Expert Mode
- Setup Wizard
- Systemcheck / Preflight
- Keine Hardcoded Secrets
- Immer .env.example pflegen
- README aktuell halten

## Support-Konzept

Alle Projekte bleiben kostenlos.

Es darf ein dezenter gemeinsamer Support-Link eingebaut werden:

- README
- Footer
- About-Seite
- Dokumentation

Keine Pop-ups.
Keine Paywall.
Keine aggressiven Spendenhinweise.
Keine gesperrten Funktionen für Nichtspender.

## Arbeitsregel für Claude

Claude setzt immer nur den von Nova definierten nächsten Schritt um.

Keine großen Architekturänderungen ohne Nova.
Keine neuen Frameworks ohne Begründung.
Keine unnötigen Komplett-Umbauten.

## Pflicht nach jedem Schritt

Am Ende immer eine Rückmeldung an Nova liefern:

1. Was wurde umgesetzt?
2. Welche Dateien wurden geändert?
3. Welche Probleme gab es?
4. Welche offenen Punkte bestehen?
5. Welche Risiken bestehen?
6. Welche Tests wurden durchgeführt?
7. Wurde README/Doku angepasst?
8. Empfehlung für den nächsten Schritt
