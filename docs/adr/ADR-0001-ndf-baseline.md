# ADR-0001: NDF Baseline for CastCore

## Status

Accepted

## Kontext

CastCore soll als erstes echtes Referenzprojekt nach dem Nova Development Framework geführt werden.

## Entscheidung

Eine NDF-Baseline wird in CastCore eingeführt.

## Konsequenzen

- CastCore erhält Project System, Project Brain, Quality Gates und Work Package Queue.
- NDF kann an einem realen Projekt validiert werden.
- Zukünftige Änderungen werden strukturierter planbar.
