# CastCore NDF Workflow

## Standardablauf

```text
Nova plant und reviewed
Claude bearbeitet kleine Work Packages
Kay prüft in VS Code/GitHub Desktop
Kay committet und pusht
Nova bewertet Rückmeldung und nächsten Schritt
```

## Rollen

### Kay

- entscheidet
- prüft Änderungen
- committet
- pusht
- gibt Nova Feedback

### Nova

- plant Architektur
- erstellt Prompts
- bewertet Risiken
- reviewed Rückmeldungen
- priorisiert Work Packages

### Claude

- bearbeitet konkrete Work Packages
- ändert nur erlaubte Dateien
- führt keine Git-Veröffentlichung durch
- liefert Rückmeldung an Nova

## Regel

Kein Work Package ohne klare Grenzen.
