# Claude Prompt – CC-WP-006 Security Baseline Review

## Rolle

Du bist Claude und prüfst CastCore auf eine erste Security-Baseline.

## Ziel

Erstelle eine erste Security-Bewertung ohne Codeänderungen.

## Aufgabe

1. Prüfe offensichtliche Security-Dokumentation.
2. Prüfe mögliche Secret-Risiken.
3. Prüfe Docker-/Deployment-Risiken auf Dokumentationsebene.
4. Prüfe Auth-/Admin-Themen, falls vorhanden.
5. Erstelle eine Liste empfohlener Security Work Packages.

## Grenzen

- Keine Codeänderungen.
- Keine Secrets ausgeben.
- Kein Commit.
- Kein Push.

## Rückmeldung an Nova

Nutze das Standardformat.
