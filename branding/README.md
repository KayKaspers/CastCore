# CastCore Corporate Design Package

## Brand
Name: CastCore  
Claim: Stream. Manage. Control.  
Positioning: Self-hosted streaming control suite for FFmpeg, media sources, platforms and monitoring.

## Logo Concept
The CastCore mark combines a playback/control screen, cast signal pixels and a modular core shape. It is designed for dark admin interfaces, dashboards, favicons and app icons.

## Colors
- Core Green: #22C55E
- Signal Cyan: #06B6D4
- Control Blue: #3B82F6
- Deep Navy: #07111F
- Panel Navy: #111C2E
- Slate: #64748B
- Mist: #E2E8F0
- White: #FFFFFF

## Typography
Recommended: Inter. Fallback: system UI / Segoe UI / Arial / sans-serif.
Do not bundle font files unless the license is checked separately.

## Usage
Use the dark horizontal logo for product headers and dashboards. Use the icon-only version for favicon, app icon and sidebar collapsed states. Prefer dark backgrounds with green/cyan accents.
